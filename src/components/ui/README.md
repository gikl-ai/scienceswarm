# src/components/ui

Layer 0 and Layer 1 of the design system's component stack.

- **Layer 0** — raw shadcn/ui primitives. Files named by the primitive
  (`button.tsx`, `dialog.tsx`, `command.tsx`, …). Generated by
  `npx shadcn@latest add <name>`.
- **Layer 1** — ScienceSwarm wrappers. Files named `ss-<primitive>.tsx`.
  Each wrapper pins the design tokens, density defaults, and microcopy
  expected throughout the app.

Application code imports the `ss-*` wrappers only. Raw shadcn imports
from `src/app/**` or `src/components/research/**` are caught by the
drift-check (`scripts/check-design-drift.ts`).

The wrapper layer lands in a dedicated PR after this foundation merges.
