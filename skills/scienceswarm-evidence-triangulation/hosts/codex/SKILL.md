# ScienceSwarm Evidence Triangulation

Use this skill when you need a disciplined evidence review before making a
technical decision, writing code against a standard, or trusting a scientific
claim.

## Workflow

1. State the claim precisely.
2. Collect at least three independent primary or official sources.
3. Summarize agreement, disagreement, and caveats without collapsing them.
4. Mark each conclusion as fact or inference.
5. Recommend the next action proportional to the remaining uncertainty.

## Output shape

- `Claim`
- `Agreement`
- `Disagreement`
- `Inference vs fact`
- `Recommendation`
